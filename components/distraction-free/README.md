DistractionFree
===============

### Import
```js
  import DistractionFree from '@govuk-frederic/distraction-free';
```
<!-- STORY -->

### Usage

Simple
```jsx
<DistractionFree onClick={(e) => {}}>
   content goes here
</DistractionFree>
```

With title
```jsx
<DistractionFree arrowLeftTitle="Back to example" onClick={(e) => {}}>
   content goes here
</DistractionFree>
```
